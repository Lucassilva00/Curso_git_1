# funciona como h1 do html, ## h2 e assim por diante.
negrito = __palavra__ ou **palavra**
itálico = _palavra_ ou *palavra*

lista sem ordenação = * item
lista om ordenação = 1. item
É possível fazer sublista nas lista, apenas colocando a sublista identada na lista
ex: 1. item
        1. sub_item

adicionar uma imagem localmente ou externa:
![(desc da img)alt da imagem](caminho_da_imagem/link_da_imagem)

adicionar links ao projeto:
[Texto do link](link)
OBS.: se for link do github pode inserir diretamente sem colchetes ou parenteses
é possível fazer uma juncão entre link na img

[![desc img](caminho_da_imagem/link_da_imagem)](link)

Adicionar código ao readme
'''linguagem_usada
código
